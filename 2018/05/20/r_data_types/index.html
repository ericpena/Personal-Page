R Data Types

Learning the basics of R data types

Published Sun, May 20, 2018 by Eric Peña

Vectors

All elements in an R vector must have the same mode: integer, numeric, character, logical, complex, etc.

x <- c(88, 12, 23, 74)

Find the length of a vector with length(x):

length(x)

## [1] 4

To iterate through the vector requires an index:

second <- function(x) {
  for (i in 1:length(x)) {
    if(i == 2) break # break out of loop
  }
  return(x[i])
}
second(x)

## [1] 12

Matricies

One way to create a matrix:

y <- matrix(c(1,2,3,4), nrow = 2, ncol = 2)

or simply:

y <- matrix(c(1,2,3,4), nrow = 2)
y

##      [,1] [,2]
## [1,]    1    3
## [2,]    2    4

Using the byrow argument (default = FALSE):

m <- matrix(c(1,2,3,4,5,6), nrow = 2, byrow = T)
m

##      [,1] [,2] [,3]
## [1,]    1    2    3
## [2,]    4    5    6

To retrieve information from a matrix:

m[,2]

## [1] 2 5

m[2,]

## [1] 4 5 6

m[2,2]

## [1] 5
